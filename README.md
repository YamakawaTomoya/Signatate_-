# Signatate_練習問題「お弁当の需要予測」

Signateの練習問題「お弁当の需要予測」について、需要数の予測モデル(重回帰モデル)を構築しました。
評価指標はRMSEで、最終スコアは 11 でした。

##参考にした書籍
⓵「機械学習のための特徴量エンジニアリング その原理とPythonによる実践」
（Alice Zheng 、Amanda Casari 著、株式会社ホクソエム 訳）
⓶「事例で学ぶ特徴量エンジニアリング」
（Sinan Ozdemir　著、田村 広平、大野 真一朗　監訳、砂長谷 健、土井 健、大貫 峻平、石山 将成　訳）

###全体方針
1.探索的データ分析(EDA)を行う。
2.各特徴量に4つの尺度を当てる。
3.特徴量エンジニアリング⓵:特徴量改善を行う。
4.特徴量エンジニアリング⓶:特徴量作成を行う。
5.特徴量エンジニアリング⓷:特徴量選択を行う。
6.訓練を通じ、特徴量エンジニアリングとモデル選択が正しいかを評価する。
7.テストデータの特徴量に同じ処理を施し、予測する。
8.予測結果をcsvへ出力する。

####使用技術：Python、pandas、scikit-learn
